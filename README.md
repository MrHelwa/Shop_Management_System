this is demo for an online shop management system
it uses sql server for the database
just plug and play.
git colne put_github_link_here.
open the .sln file.
click on the play button on the top right for vs code 
the connection string uses sql server express.
if you have sql server developer user this connection string. "Server=.\SQLEXPRESS;Database=ProductShopDB_ITP;Integrated Security=true;TrustServerCertificate=true;".
you will find this connection string  in "Models" folder => "ShopDbContext.cs" file. replace existing string with the above one.
enjoy.
